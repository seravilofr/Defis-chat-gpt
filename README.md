This repository is meant to gather challenges I asked ChatGPT to give me, from easiest to hardest, to learn Python concepts.

The prompt in French is *"Sur cette page je vais te demander des défis pour apprendre python. Je suis débutant et je ne connais rien à Python donc chaque défi se concentrera sur un concept en partant de concepts simples et en allant de plus en plus compliqué. Donne moi mon premier défi et dis moi quel concept ça me permet d'apprendre"*
